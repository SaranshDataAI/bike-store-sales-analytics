# Bike Store Sales Analytics Dashboard

An end-to-end data analytics project analyzing bike store sales performance using SQL and Power BI.

## Project Overview

This project explores sales data from a fictional bike store database to uncover insights about:

- Revenue trends
- Product performance
- Brand performance
- Store performance
- Inventory levels

The results are visualized in an interactive Power BI dashboard.

---

## Tools Used

- PostgreSQL
- SQL
- Power BI
- DAX

---

## Key Business Questions

1. Which products generate the most revenue?
2. Which brands drive the most sales?
3. Which stores perform best?
4. How does revenue change over time?
5. What inventory levels exist across categories?

## Key Insights

- Trek generated the highest revenue (~$4.6M)
- Mountain Bikes are the top-selling category
- Baldwin Bikes is the best-performing store
- Cruiser Bikes hold the largest inventory

---

## Dataset

The dataset includes sales, product, inventory, and customer information across multiple bike stores.

Tables used:

- customers
- orders
- order_items
- products
- brands
- categories
- stocks
- stores
